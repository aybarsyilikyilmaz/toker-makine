# Toker Teknik Makina ve Bakım Hizmetleri

Toker Teknik Makina kurumsal web sitesi. Talaşlı imalat, makine bakım ve onarım süreçlerini en yüksek profesyonellikle sergileyen modern bir Next.js uygulamasıdır.

## Özellikler
- **Talaşlı İmalat:** Torna, freze ve planya süreçlerinin detaylı sunumu.
- **Dinamik Makine Parkuru:** Teknik detaylar ve yüksek çözünürlüklü görseller.
- **Kurumsal Kimlik:** SEO uyumlu meta veriler ve modern endüstriyel tasarım.
- **İletişim ve Koordinat:** Gebze Hasköy Sanayi Sitesi lokasyon ve doğrudan erişim.

## Teknik Altyapı
- **Framework:** Next.js 15+ (App Router)
- **Stil:** CSS Modules & Vanilla CSS
- **İkonlar:** Lucide React
- **Performans:** Yüksek hız ve mobil uyumluluk odağı.
